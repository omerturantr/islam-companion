# Islam (Expo)

An Expo React Native app focused on prayer times, Quran reading, Qibla, and daily worship tools.

## Features
- Prayer times with reminders and location support
- Quran reader with audio, bookmarks, and notes
- Qibla compass
- Duas, tasbih counter, zakat calculator, and notes

## Tech
- Expo + React Native + TypeScript
- React Navigation
- AsyncStorage for local data

## Setup
1) Install dependencies:
   `npm install`
2) Start the app:
   `npm run start`

## Environment
If you use environment variables, add them to a local `.env` file (ignored by git).

## Scripts
- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
